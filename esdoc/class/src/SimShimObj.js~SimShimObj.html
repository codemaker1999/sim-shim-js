    SimShimObj

    Responsible for creating ThreeJS Objects and setting up update functions for plots.

    Advanced usage might require access to the input manifest, ThreeJS object, or update function. These are stored as instance varibales:

    • the input manifest (if present) is stored as ssObj.manifest
    • the ThreeJS object is stored as ssObj.threeObj
    • the update function is stored as ssObj.update
